Commands

Bot Commands

All available slash commands

/play
Start playing Holy Quran Radio in your voice channel
/stop
Stop the radio and disconnect from voice channel
/stations
View all available radio stations
/nowplaying
Show the currently playing station
/volume [level]
Adjust the playback volume (1-100)
/kick [member] [reason]
Kick a member from the server
/ban [member] [reason]
Ban a member from the server
/unban [user_id]
Unban a user from the server
/timeout [member] [duration] [reason]
Timeout a member for specified minutes
/warn [member] [reason]
Issue a warning to a member
/warnings [member]
View warnings for a member
/clear [amount]
Delete messages from the channel
/slowmode [seconds]
Set slowmode for the channel
/lock
Lock the current channel
/unlock
Unlock the current channel
/automod [feature] [enabled]
Toggle AutoMod features (antispam, antilinks, anticaps, antiinvites, antimentions)
/automod-status
View current AutoMod configuration
/setlogchannel [channel]
Set the channel for AutoMod logs
/serverinfo
Display server information
/userinfo [member]
Display user information
/avatar [member]
View a user's avatar
/ping
Check bot latency
/invite
Get the bot invite link
/help
View all commands
